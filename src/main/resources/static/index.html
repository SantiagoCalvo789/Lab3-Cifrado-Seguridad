Crypto Workspace

Ingresa la seed y procesa tu texto.

Esta vista usa los endpoints del backend que ya tienes en Spring Boot. Puedes pegar texto cifrado en Base64 para descifrarlo o texto plano para cifrarlo con la misma seed.

Blowfish Console

Cifrar o descifrar desde el navegador

La misma pantalla sirve para probar ambos flujos. Si vas a descifrar, pega aquí el texto Base64 que te devolvió el backend.

Usa exactamente la misma seed con la que se cifró el mensaje.
`Descifrar` espera Base64. `Cifrar` espera texto plano.

Cómo usarlo

  • Escribe la seed.
  • Pega el texto cifrado o el texto plano.
  • Presiona Descifrar o Cifrar.

Pistas rápidas

Si el descifrado falla, normalmente es porque la seed no coincide o porque el valor de entrada no es un Base64 válido.

Endpoint usado

POST /crypto/encrypt

POST /crypto/decrypt
